# E-Commerce Backend

## Overview
This is the backend of an E-Commerce Web Application developed using Spring Boot. It provides REST APIs for managing products and supports CRUD operations.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- Maven
- MySQL

## Features
- Add Product
- View Products
- Update Product
- Delete Product
- Product Image Support
- REST API Architecture

## Project Structure

src/main/java
├── controller
├── service
├── repo
├── model

## API Endpoints

GET /products
POST /product
PUT /product/{id}
DELETE /product/{id}

## How to Run

1. Clone the repository
2. Configure MySQL database
3. Update application.properties
4. Run the Spring Boot application
